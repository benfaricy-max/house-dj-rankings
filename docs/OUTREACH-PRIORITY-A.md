# Priority-A Outreach — ready to send

> Companion to `GTM.md` (§2 templates) and `docs/FEE-ANCHORS-WORKSHEET.md`. Ten
> strong-buy/buy reads, each with a cold email + a DM, built from live index data
> (fee band, demand-implied band, gap %, recent booking regions). **One rule from
> GTM.md: specific person, specific artist, specific number.** The artist and number
> are done — the only blank is `[name]`, the real buyer weighing this act. Fill it or
> don't send. Generate a Pitch Link (`#/value/<slug>`, 7-day expiry) and paste it.
>
> Voice check (BRAND.md): terminal-meets-warehouse. No "🔥", no "blowing up", no
> exclamation marks. The number is the hook; let it sit there.

---

## How to use

1. Pick the act whose buyer you can actually name.
2. Open the Value Gap report, copy a **Pitch Link** framed "Buying (offer case)".
3. Paste it where each template says `[Pitch Link]`.
4. End every reply/call with the anchor ask (worksheet): *"what did the last
   comparable booking actually cost?"*

Suggested buyer archetype is noted per act — but a real named person beats the
archetype every time.
